How I Built a CI/CD Pipeline Using Jenkins, Docker & AWS

May 2026 5 min read

In this project, I designed and implemented a complete CI/CD pipeline using Jenkins, Docker, GitHub, and AWS EC2.

The primary goal was to automate application build, testing, and deployment processes to reduce manual effort and improve deployment reliability.

Technologies Used

  • Jenkins
  • Docker
  • AWS EC2
  • GitHub
  • Linux

Project Workflow

The pipeline automatically pulls source code from GitHub whenever changes are pushed. Jenkins then builds the application, creates Docker images, and deploys containers on AWS EC2 instances.

This automation improved deployment speed and reduced manual deployment errors significantly.

Key Learnings

  • CI/CD automation using Jenkins
  • Docker containerization
  • AWS EC2 deployment
  • Linux server management
  • GitHub integration

This project helped me gain hands-on experience in DevOps automation, containerization, and cloud deployment strategies.
